KiritoriMage - A Simple Image File Splitter
===========================================

What's this?
-------------
A tool to split an image into multiple parts.

Environment
------------
* .NET Framework 4.0

How to use?
------------
1. Drag and drop an image file.
   (The command-line argument also supported, like: KiritoriMage c:\temp\foobar.jpg)

2. Left-click on the horizontal/vertical bar to specify dividing lines.

3. Left-click on the crossing area within the lines to specify regions.
  (You can cancel with right-clicking.)

4. Click "Save" button to save {image}-000.jpg, -001.jpg, -002.jpg.. files.
